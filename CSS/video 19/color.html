About Color

color can be represented in :

  1. Color keywords
  2. Hex color code
  3. RGB
  4. RGBA(here 'a' stand for ALPHA ALPHA means opesity of color.Range of ALPHA is 0 to 1 )
  5. HSL(here 'h' stand for hue,'s' stand for saturation,'l' stand for lightness)
  6. HSLA

This all is also apply on background color,border color e.t.c.
